# VGA API for the ARM STM32F4

This API allows the user to post the following items on a VGA display:

### shapes
* Line
* Rectangle
* Triangle
* Ellipse

### Other features
* Display text      
*[multiple fonts, bold, italic, full ASCI]*
* Display bitmaps   
*[basic bitmaps are included in API]*
* Clear screen
